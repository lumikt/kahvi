Latex Reference App

Welcome to Latex reference app!
Here you can create references that we'll transform to bibtex form for you.

To create references, work as follows:
1. Go to "Add new reference"
2. Add all the information about your reference
3. After saving the reference you can see, edit and delete your references from "Added references"
4. Go to "Added references in bibtex form" to download your references as bibtex file
